Glossary

Alternative An object with specified attributes or attribute values.
Add alternatives to the project if you wish to limit results to specified objects. If Alternatives are not specified, any combination of attributes or attribute values may be the preferred result.
Alternative Value The combination of attributes or attribute values that make up an alternative.
Attribute Characteristics that describe the item in question.
e.g. Item: Car - Attributes: Color,Number of Doors,Car Type
Attribute Domain Values that an attribute can take on.
e.g. Attribute: Color - Domain: Red,Blue,Yellow
Consistent No attribute is preferred to itself. A project is inconsistent if there exist any cycles in the preference graph.
Dominance An alternative dominates another alternative if it is preferred based on the input preferences.
Role The title of a person or of a group of people which are included in the preference making process.
Role Hierarchy A graphical representation in which Roles are ranked above the other according to status or authority.
Stakeholder A member of a certain Role. Each stakeholder contributes their own preferences.
