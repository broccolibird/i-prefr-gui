    • Controls



    Stakeholder Drop-down Menu New preferences will be attached to the Stakeholder shown in the drop-down menu
    Save Save preferences to the current Stakeholder's preference file. A file chooser will appear to create a new preference file if the Stakeholder does not yet have one.
    Save As Save preferences to a new Stakeholder preference file. After selecting the file name from the file chooser, the file will be attached to the current Stakeholder.
    Load Existing File Loads an existing preference file and attaches it to the current Stakeholder.
    Clear Removes all preferences from the entry fields.
