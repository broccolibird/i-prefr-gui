Alternative Values

If you chose Use Entire Possible Alternative Space from the Add Alternatives menu, continue to the next step.

If you chose to Specify Specific Alternatives, you will now be able to set values for each alternative.


Setting Alternative Values

Use the drop-down boxes to the right of each of the alternatives to set its value.

If there are no drop-down boxes to the right of the alternative, verify that attributes and attribute domains have been added to the project.
