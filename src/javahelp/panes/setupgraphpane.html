Setup Preferences (TCP-Net)

Add Attributes to the Preference Map

In order to add an Attribute to the Preference Map, start by verifying that the graph is in Editing mode. Then, select the Attribute to be added from the Attribute drop-down menu. Click on the graph view to place the Attribute in the Preference Map.

Adding Preferences to Edges

After adding two or more Attributes to the graph, edges between Attributes can be added by selecting (while still in Editing mode) an Attribute and dragging the cursor to another Attribute. After adding an edge, preferences can be added by switching to Annotating mode and selecting an edge. On selecting the edge, a dialog will appear.


Select the field labeled "click to enter ConditionMap" and another dialog will appear.

Create the condition under which the preference exists and press OK. See Creating Condition Statements for more help on this step. After adding a condition, finish the preference by selecting a preference statement.

Finish adding the preference by selecting OK on the original dialog. The preference should now be visible on the graph view.

Adding Preferences to Nodes

After adding one or more Attributes to the graph, you can start creating preferences. While in Annotating mode select a node and then right click on it. A Dialog will appear.


Select the field labeled "click to enter ConditionMap" and another dialog will appear.

Create the condition under which the preference exists and press OK. See Creating Condition Statements for more help on this step. After adding a condition, finish the preference by selecting "Click to enter DomainPreferenceMap" and filling out the domain preference dialog.

Finish adding the preference by selecting OK on the original dialog.

Please note that preferences added to nodes will not be visible in the graph view.
