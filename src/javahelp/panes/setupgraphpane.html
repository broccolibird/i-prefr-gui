Setup Preferences (TCP-Net)

Add Attributes to the Preference Map

In order to add an Attribute to the Preference Map, start by verifying that the graph is in Editing mode. Then, select the Attribute to be added from the Attribute drop-down menu. Click on the graph view to place the Attribute in the Preference Map.

Create Preferences

After adding two or more Attributes to the graph, you can start adding Attribute preferences. While still in Editing mode, click on an Attribute and drag the cursor to another Attribute. Continue adding Attributes and relationships until you have finished the Role Hierarchy.
