Setup Preferences (TCP-Net)

Controls

Attribute

Select an Attribute from the drop-down box to add to the Preference graph


Mode

EDITING Add Attributes to the preference graph and create links between Attributes to create relationships.
TRANSFORMING Click and drag the entire viewing window to focus on different areas.
PICKING Select an Attribute and drag it to the desired location within the graph view.
ANNOTATING Select an Attribute or an edge in order to add a preference condition.

Zoom

Use the zoom controls to increase (+) or decrease (-) the size of the graph.
