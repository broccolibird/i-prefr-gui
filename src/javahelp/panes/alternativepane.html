Add Alternatives


Use Entire Possible Alternative Space

This is the default option and should be used when you want to find the most preferred solution over all combinations of Attribute values.

If you would like to find the most preferred solution over a specific set of Attribute value combinations, please see Specify Specific Alternatives below.


Specify Specific Alternatives

Select the Specify Specific Alternatives button with your mouse or press Alt+p on your keyboard.

Add the first Alternative to the project by entering the new Alternative's name into the Alternative Name field.

Add additional Alternative fields by selecting the plus button or by pressing the enter key on your keyboard.

Remove Alternatives

Alternatives can be removed from the project by selecting the x button to the right of the Alternative to be removed.
